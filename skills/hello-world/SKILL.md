# Hello World

A minimal skill used to verify that skills from this repository install and trigger correctly.

## What to do

When this skill triggers, respond with a single short greeting that confirms the skill ran:

```
Hello from the hello-world skill — install successful.
```

Do not add extra commentary, summaries, or follow-up questions. The point of this skill is to be a clear, unambiguous signal that the install and triggering pipeline is working end to end.
